# Bright Smile Dental Clinic Website Redesign

## Future Interns – UI/UX Design Task 1 (2026)

### Project Overview
This project is a UI/UX redesign of a local dental clinic website created for the Future Interns UI/UX Design Task 1 (2026). The objective was to design a clean, modern, and responsive website that improves user experience and encourages visitors to book appointments.

### Business Type
Local Dental Clinic

### Objective
- Create a clean and modern interface.
- Improve navigation and readability.
- Increase appointment bookings.
- Design a responsive website for desktop and mobile.

### Pages Designed
- Homepage
- Service Detail Page
- Contact Page

### Key Features
- Modern UI Design
- Responsive Layout
- Appointment Booking CTA
- Contact Form
- Mobile-Friendly Navigation

### Tools Used
- Figma
